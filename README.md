### Interactive Simulations

## ğŸš€ Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Local web server (optional, for best experience)

### Installation

1. Clone or download the project:
```bash
git clone <repository-url>
cd NSD_project
```

2. Open with a local server (recommended):
```bash
# Using Python
python -m http.server 8000

# Using Node.js http-server
npx http-server

# Using PHP
php -S localhost:8000
```

3. Or simply open `main.html` in your web browser

4. Navigate through the site:
   - Start at `main.html` for the welcome page
   - Visit `Laws.html` to browse all physics laws
   - Click on any law to see detailed explanations
   - Access interactive simulations via animation links
